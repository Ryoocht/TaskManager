# About TaskManager
A simple web application to manage tasks [Todo list app].

## Prerequisites
- Java 15
- Maven
- Eclipse

### Plugins / Dependencies
- Framework: Springframework
  - thymeleaf
  - mybatis
  - devtools
  - jquery
- Database: MySQL
